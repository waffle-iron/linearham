Detailed Description

Some simple linear algebra routines.

Note that we universally use Eigen::Ref<X> here, which means that we can pass in an X or a Block of an X. See the section on Ref in https://eigen.tuxfamily.org/dox/TopicFunctionTakingEigenTypes.html If you want it to be const, it's essential to use const Eigen::Ref<const X>& (don't forget the ampersand!)
